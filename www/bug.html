New Features and Bug Fixes in SPPARKS

This page lists new features and bug fixes in the SPPARKS simulator since the current version was released.

To add these features/fixes to your version of SPPARKS, download the current tarball from the download page, which includes all the new features/fixes listed below, and rebuild the code.

The current version of SPPARKS is 11 Nov 2009.

11 Nov 2009

New release of SPPARKS. 22,790 lines of code in src dir.

These are the most significant features added or enhanced since the 13 Apr 2009 version. People who contributed to the features are listed in parentheses:

  • app_style diffusion (Greg Wagner and Xiaowang Zhou, Sandia)
  • app_style relax
  • barrier (Ed Webb, Sandia)
  • create_box
  • create_sites
  • deposition (Ed Webb, Sandia)
  • diag_style propensity
  • dimension
  • dump_modify
  • dump_one
  • lattice
  • pair_coeff
  • pair_style
  • processors
  • read_sites
  • region
  • Schwoebel hops in app_style diffusion
  • set
  • undump

Descriptions of all these enhancements and other bug fixes can be viewed on this page.

13 April 2009

Initial public release of SPPARKS. 17,341 lines of code in src dir. This version supercedes earlier distributed beta versions.
