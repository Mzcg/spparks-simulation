New Features and Bug Fixes in SPPARKS :h3

This page lists new features and bug fixes in the SPPARKS simulator
since the current version was released.

To add these features/fixes to your version of SPPARKS, download the
current tarball from the "download page"_../download.html, which
includes all the new features/fixes listed below, and rebuild the
code.

The current version of SPPARKS is 13 April 2009.

:line

[7 Jul 2009]

Added a "dump_one"_doc/dump_one.html command to force a shapshot of
the lattice to be written to the file specified by the
"dump"_doc/dump.html command.

This is the "list of changed files"_patches/files.7Jul09 from the 26
Jun 2009 version.

:line

[26 Jun 2009]

Aidan Thompson (Sandia) fixed a bug in the recent "diag_style
cluster"_doc/diag_cluster.html upgrade.

This is the "list of changed files"_patches/files.26Jun09 from the 24
Jun 2009 version.

:line

[24 Jun 2009]

Enhanced the "pin"_doc/pin.html command for use by the "app_style
potts/pin"_doc/app_potts_pin.html command.  Also enhanced the grain
identification performed by the "diag_style
cluster"_doc/diag_cluster.html command to not include pinned sites.

This is the "list of changed files"_patches/files.24Jun09 from the 12
May 2009 version.

:line

[12 May 2009]

Fixed a dumb bug with the rejection method (rKMC) applied to
"app_style diffusion"_doc/app_diffusion.html.

This is the "list of changed files"_patches/files.12May09 from the 14
Apr 2009 version.

Thanks to Greg Wagner (Sandia) for identifying the problem.

:line

[14 Apr 2009]

Removed an application from the distribution that is still
a work-in-progress.  Wasn't meant to be released yet.

This is the "list of changed files"_patches/files.14Apr09 from the 13
Apr 2009 version.

:line

[13 April 2009]

Initial public release of SPPARKS.  17,341 lines of code in src dir.
This version supercedes earlier distributed beta versions.
