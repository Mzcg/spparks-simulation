app_style command

Syntax: 

app_style style args
  • style = application style name
  • args = args

Examples: 

app_style ising 100 100
app_style potts 1000 1000 4

Description:

diag_style style delta arg option arg option ...

Run a diagnostic function called style after every delta time units. The option logfreq is available for all diag_styles and requires two arguments, just like in the stats command.

The diagnostic functions available are:

cluster cluster3d cluster2d eprof3d energy energy2d energy3d

These diagnostics are only available for app_styles derived from AppLattice, AppLattice2d, AppLattice3d, as indicated by the suffix.

The energy diagnostic computes the total energy for all sites and uses it for stats output.

Restrictions: none

Related commands:
