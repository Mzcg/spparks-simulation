diag_style deposition command

Syntax: 

diag_style deposition keyword value keyword value ...
  • deposition = style name of this diagnostic
  • see the diag_style command for additional keywords that can be appended to a diagnostic command

Examples: 

diag_style deposition

Description:

The deposition diagnostic calculates counts of the total cummulative number of successful and failed deposition events since the simulation began. These values are printed as stats output via the stats command.

A successful deposition event is one that resulted in an atom added to the lattice. A failed deposition event is one that was attempted but no suitable site could be found and thus no atom was added.

Restrictions:

This diagnostic can only be used with the app_style diffusion2 application.

Related commands:

diag_style, stats

Default: none
