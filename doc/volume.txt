:line

volume command :h3

[Syntax:]

volume V :pre

V = volume of system (liters) :ul

[Examples:]

volume 1.0e-10 :pre

[Description:]

This command sets the volume of the system used by the "app_style
chemistry"_app_chemistry.html application for performing chemical
reactions.  For example, if a Gillespie-style simulation is performed
for a biological cell, it would be the volume of the cell.

[Restrictions:]

This command can only be used as part of the "app_style
chemistry"_app_chemistry.html application.

[Related commands:] none

[Default:] none
