app_style command

Syntax: 

app_style style args
  • style = application style name
  • args = args

Examples: 

app_style ising 100 100
app_style potts 1000 1000 4

Description:

app_style test/group app_style test/group2

run an app to test group solver performance two versions of this app group = Alex's original version group2 = Steve's trimmed down version (used for JCP paper)

also need to define a "solve" style: see "* solve_style" below

Extra commands supported by this app:

event 200 30 1 Defines properties of the event table: 1. Size of event table 2. Max number of dependencies 3. Size of propensity adjustment (%) 4. optional Low memory switch to generate dependencies randomly on the fly instead of storing the event table. 5. Random seed for the low_mem option.

Examples: event 200 30 1 event 200 30 1 lo_mem 384745

run 0.5 run for time (seconds)

Restrictions: none
