volume command

Syntax: 

volume V
  • V = volume of system (liters)

Examples: 

volume 1.0e-10

Description:

This command sets the volume of the system used by the app_style chemistry application for performing chemical reactions. For example, if a Gillespie-style simulation is performed for a biological cell, it would be the volume of the cell.

Restrictions:

This command can only be used as part of the app_style chemistry application.

Related commands: none

Default: none
