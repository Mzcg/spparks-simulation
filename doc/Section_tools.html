7. Additional tools

KMC is designed to be a computational kernel for performing kinetic Monte Carlo computations. Additional pre- and post-processing steps are often necessary to setup and analyze a simulation. This section describes additional tools provided with the KMC distribution.

Our group has also written and released a separate toolkit called Pizza.py which provides tools for doing setup, analysis, plotting, and visualization for KMC simulations. Pizza.py is written in Python and is available for download from the Pizza.py WWW site.

Note that many users write their own setup or analysis tools or use other existing codes and convert their output to a KMC input format or vice versa. The tools listed here are included in the KMC distribution as examples of auxiliary tools. Some of them are not actively supported by Sandia, as they were contributed by KMC users. If you have problems using them, we can direct you to the authors.

The source code for each of these codes is in the tools sub-directory of the KMC distribution. There is a Makefile (which you may need to edit for your platform) which will build several of the tools which reside in that directory. Some of them are larger packages in their own sub-directories with their own Makefiles.
