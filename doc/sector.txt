:line

sector command :h3

[Syntax:]

sector flag keyword value ... :pre

flag = {yes} or {no} :ulb,l
zero or more keyword/value pairs may be appended :l
keyword = {nstop} or {tstop} :l
  {nstop} value = N
    N = 
  {tstop} value = dt
    dt = :pre
:ule

[Examples:]

sector no
sector yes
sector yes nstop 1.5
sector yes tstop 5.0 :pre

[Description:]

This command sets the molecular count of a chemical species for use in
the "app_style chemistry"_app_chemistry.html application.

The species ID can be any string defined by the
"add_species"_add_species.html command.

[Restrictions:]

This command can only be used as part of on-lattice applications
as specified by the "app_style"_app_style.html command.

[Related commands:]

"app_style"_app_style.html

[Default:]

The option defaults are nstop = 1.0.
