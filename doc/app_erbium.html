app_style erbium command

Syntax: 

app_style erbium frac seed keyword values ...
  • erbium = application style name
  • frac = fraction of tetrahedral sites that will initially be hydrogen
  • seed = random number seed (positive integer)
  • see the app_style command for additional keywords that can be appended

Examples: 

app_style erbium 0.1 12345 lattice file tmp.lattice site 2 0

Description:

This application simulates a model on a 3-fold lattice which consists of an fcc lattice with additional tetrahedral and octahedral interstitial sites. The 3-fold lattice should be created using the erbium_lattice.py tool in the tools directory and read in by the lattice keyword of the app_style commmand.

This application initializes sites on the lattice in the following way. All fcc sites are for Erbium atoms. All octahedral sites are initially vacant. A fraction of the tetrahedral sites, as specified by frac, are initialized as Hydrogen atoms; the remainder are vacant.

The event command is used to define what kinds of diffusive hops occur in the model. These can include correlated hops where a central site coordinates a change of atoms on two of its neighbors. Events that transmute a Hydrogen atom to a Helium are also possible.

The following additional commands are defined by this application.

dump output of lattice snapshots
event definition of an event on the 3-fold lattice
stats output of system info
temperature set Monte Carlo temperature

Restrictions: none

Related commands: none

Default: none
